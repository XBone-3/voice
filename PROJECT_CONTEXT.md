Project Name

Nova

Description

Nova is not a chatbot.

Nova is an Android-native intelligence layer.

Nova is always awake.

Nova listens continuously.

Nova understands context.

Nova performs Android actions.

Nova speaks naturally.

Nova can automate Android.

Nova should eventually become the user's primary interface with Android.

Core Philosophy

Less Interface.

More Intelligence.

The user should spend less time navigating applications and more time talking naturally.

The assistant should reduce cognitive load.
